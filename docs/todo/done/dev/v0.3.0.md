# v0.3.0 — 协议详情视图（Protocol Detail View）

## 协议元数据展示

- [ ] 显示协议名称、版本、默认字节序
- [ ] RFC 编号可点击跳转到 RFC 原文
- [ ] 显示描述、状态、层级、依赖列表

## 字段列表

- [ ] 按定义顺序展示所有字段（名称、类型、位宽、枚举映射）

- [ ] checksum 字段显示算法和覆盖字段
- [ ] condition 字段显示可读条件表达式（如 "when ihl > 5"）
- [ ] length_ref 字段显示引用字段名、缩放因子、偏移量
