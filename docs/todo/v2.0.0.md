# v2.0.0 — 增强交互与协作

> 里程碑：PSL UI 从查阅工具升级为协议开发协作平台。
> 前置：v1.0.0 全部完成

## 新增能力

- [ ] 在线 PSL 编辑器 — 浏览器内编写和验证 .psl 文件
- [ ] 协议对比视图 — 并排对比两个协议的字段差异
- [ ] 数据包捕获可视化 — 上传 pcap 文件，逐层解析展示
- [ ] 协议依赖关系图 — 交互式协议栈拓扑图
- [ ] 用户自定义协议上传与管理
- [ ] 协议收藏与历史记录
- [ ] 分享链接 — 编解码结果可通过 URL 分享

## 技术改进

- [ ] WebSocket 实时通信（替代轮询）
- [ ] 离线模式（Service Worker 缓存协议数据）
- [ ] 性能优化（虚拟滚动、懒加载）
